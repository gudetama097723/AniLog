# AniLog

## サイト概要
### サイトテーマ

アニメ作品の感想や評価を記録・共有できる、アニメ特化型レビューSNSサイトです。
​
### テーマを選んだ理由

私は以前からアニメ鑑賞が趣味で、多くの作品を視聴してきました。
しかし、視聴作品が増えるにつれて「どの作品のどの話数が特に面白かったか」「どの作品のOPや演出が印象的だったか」など、自分の感想や記憶を整理することが難しいと感じるようになりました。

また、既存のレビューサイトでは作品全体の評価を見ることはできても、話数ごとの感想や細かな魅力、ユーザー同士の交流を深く行えるサービスは少ないと感じています。
特にアニメを見始めたばかりの人にとっては、「何を見ればよいのか分からない」「実際に見た人のリアルな感想を知りたい」と感じる場面も多いと考えました。

そこで、自分の視聴記録を日記のように管理できるだけでなく、他ユーザーと感想を共有したり、ランキングやレビューを通じて新たな作品と出会えたりするサービスを制作したいと考え、このテーマを選びました。

### ターゲットユーザ
- アニメの感想や視聴記録をまとめたい人
- 他ユーザーのレビューを参考にして作品を探したい人
- 好きな作品について他の人と語り合いたい人
- これからアニメを見始めたい初心者の人

​
### 主な利用シーン
- アニメを見終わった後に感想を記録したい時
- 面白いアニメを探したい時
- 他ユーザーの評価やレビューを参考にしたい時
- 好きな作品について交流や情報共有をしたい時
